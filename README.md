# réalisation d'un site web

un site web est composé des éléments suivants :

 - html
   - balise : `<qqh>mon text<qqch>`
   - bloc head pour le titre
   - bloc body pour la page
   - div pour organiser
   - h1 ... pout les titres
   - img pour les images
   - a pour les liens
 - css
 - javascript
